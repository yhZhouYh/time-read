# time-read

> [时代书房H5版](http:www.apgbook.com/timeread/#/)

# 特色之处
1. 使用[vue-navigation](https://github.com/zack24q/vue-navigation)创建仿原生路由，记录前进后退
2. 基于cloumn布局的H5小说阅读器,自动记录阅读位置，能够调整字号和阅读背景
3. 集成微信与支付宝支付
